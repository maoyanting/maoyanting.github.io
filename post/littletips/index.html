.properties文件中定义数组

如果你在 .properties文件中定义数组：

base.module.elementToSearch=1,2,3,4,5,6

你可以在你的Java类中加载这样的数组，像这样：

  @Value("${base.module.elementToSearch}")
  private String[] elementToSearch;

参考链接：Spring .properties file: get element as an Array

ps：实测在Disconf没有用
